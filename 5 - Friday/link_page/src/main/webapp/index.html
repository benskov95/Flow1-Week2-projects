Project link page by Benjamin Skovgaard



This page contains a few links to various deployed projects, as well
as examples of usable methods on each website. The first two
use a database (running on my droplet) to retrieve their results.

GitHub link to all of the week 2 projects (and documents)

(1) Employee project from week 1 (Thursday)

(2) Bank Customer project from week 1 (Friday)

(3) HTTP project from week 2 (Wednesday)

(4) Session/cookie project from week 2 (Wednesday)


For (1): try adding the following to the end of the URL: api/employees/employee_by_id/4
For (2): try adding the following to the end of the URL: api/bankcustomer/all
For (3): try adding the following to the end of the URL: requestinfo
For (4): try adding the following to the end of the URL: sessiondemo

After filling in a name and clicking send on (5), you might have
to manually change the "SessionDemo" part of the URL to
all lowercase ("sessiondemo") for it to bring you to the correct page.

You can then try closing the tab and opening it again with
"sessiondemo" at the end of the URL to see that the server
remembers the name you sent (for a limited amount of time).
